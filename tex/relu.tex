    \section*{Definition \& Gradient}
    The Rectified Linear Unit function is defined as follows:
    \[ \begin{cases} 
        x & x > 0 \\
        0 & otherwise
     \end{cases}
  \]
  To find the gradient w.r.t. x, we use the gradient of each case. If $x > 0$, the gradient is 1. Otherwise, the gradient is 0..
